# Usage

Usage of irccd.

## Synopsis

````nohighlight
$ irccd [options...]
````

## Options

The following options are available:

  - `-c, --config file`: specify the configuration file,
  - `-f, --foreground`: do not run as a daemon,
  - `-h, --help`: show this help,
  - `-v, --verbose`: be verbose,
  - `--version`: show the version.
