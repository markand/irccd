# server-cnotice

Send a notice to a public channel. This is a notice that everyone will be notified by.

## Usage

````nohighlight
$ irccdctl server-cnotice server channel message
````

## Example

````nohighlight
$ irccdctl server-cnotice freenode #staff "Don't flood"
````
