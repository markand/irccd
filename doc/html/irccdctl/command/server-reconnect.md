# server-reconnect

Force reconnection of one or all servers.

If server is not specified, all servers will try to reconnect.

## Usage

````nohighlight
$ irccdctl server-reconnect [server]
````

## Example

````nohighlight
$ irccdctl server-reconnect
$ irccdctl server-reconnect wanadoo
````
